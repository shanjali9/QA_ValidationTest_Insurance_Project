# Insurance Data QA Validation

This project demonstrates automated data validation for an insurance dataset.

Dataset Source:
Kaggle Insurance Dataset

Tools Used:
- Python
- Pandas
- PyTest
- Jupyter Notebook

Validations Implemented:
- Schema validation
- Null value detection
- Duplicate detection
- Age range validation
- Positive charge validation
- Business logic validation (smoker costs)

Purpose:
Simulate QA validation for data pipelines similar to healthcare analytics systems.
